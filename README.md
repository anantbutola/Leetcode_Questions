# LeetCode Solutions

A personal collection of LeetCode problem solutions written in Java.

## Language

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)

## Structure

All solutions live at the root level. Each file is named using the pattern:

```
<ProblemName>_<ProblemNumber>.java
```

Each file contains the full problem statement as a block comment at the top, followed by the `Solution` class.

## Running a Solution

Each file contains a `main` method you can use as an entry point. To compile and run:

```bash
# Compile
javac <FileName>.java

# Run
java <ClassName>
```

For example:

```bash
javac RhombusSum1878.java
java RhombusSum1878
```
